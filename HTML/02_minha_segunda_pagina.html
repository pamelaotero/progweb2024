A Linguagem HTML

HTML (Abreviação para expressão inglesa Hyper Text Markup Language), que significa Linguagemde marcação de Hipertexto, é uma linguagem de marcação utilizada na construção de páginas na Web. Documentos HTML podem ser interpretados por navegadores.

Alguns recursos da linguagem HTML viwtos até o momento:

Recursosos Tags associadas Descrição
Links < a > Criar links entre páginas
Listas < ul > e < li > Criar Listas de itens ordenados
< ol > e < li > Criar Listas de itens não ordenados
Imagens < img > Inserir imagns
